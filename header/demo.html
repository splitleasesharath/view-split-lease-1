Split Lease Header Component Demo

This is a working demonstration of the reusable Split Lease header component. The header above is fully functional with all features enabled.

Try These Features:

  • Hover over "Host with Us" or "Stay with Us" to see dropdown menus
  • Click the dropdowns to toggle them
  • Try keyboard navigation (Tab, Arrow keys, Enter)
  • Resize your browser to see mobile responsive design
  • Click "Sign In" to test authentication redirect

Component Features

  • Fixed header that stays at the top while scrolling
  • Interactive dropdown menus with hover and click support
  • Fully responsive design (desktop, tablet, mobile)
  • Keyboard accessible navigation
  • Smooth animations and transitions
  • Zero external dependencies
  • Customizable via CSS variables
  • Works with any backend or framework

Quick Integration

1. Include the Files

<!-- In your <head> -->
<link rel="stylesheet" href="header.css">

<!-- Before closing </body> -->
<script src="header.js"></script>

2. Add the Header HTML

Copy the content from header.html into your page.

3. Adjust Your Content Spacing

body {
  padding-top: 72px; /* Height of fixed header */
}

Customization Options

Colors

Override CSS variables to match your brand:

:root {
  --header-bg-color: #31135D;
  --header-text-color: white;
  --button-bg: white;
  --button-text: #6D23CF;
}

Links

Edit the href attributes in the HTML to point to your pages:

<a href="/your-page">
  Your Link
</a>

Browser Support

The header component works on all modern browsers:

  • Chrome / Edge (latest)
  • Firefox (latest)
  • Safari (latest)
  • Mobile browsers (iOS/Android)
